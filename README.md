# CodeGarage
The most fast and light web browser base editor.
You can copy everything.
Also, you can save it.

## Installation
```
npm install codegarage
```

## Development
```sh
# setup
git clone git@github.com:koheishingai/CodeGarage.git
cd codegarage

# testing
node app
```
